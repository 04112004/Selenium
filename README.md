# 🌐 Selenium Automation Suite

A comprehensive Selenium-based automation project built with Python for browser automation, web interaction testing, notification systems, and WhatsApp automation.

This repository demonstrates practical implementations of automated browser workflows using Selenium WebDriver and Python scripting.

---

## 🚀 Project Overview

This project contains multiple automation scripts developed for:

- Web browser automation
- Automated notifications
- WhatsApp bot automation
- Form interaction testing
- Web data handling
- Automated browser actions

The project showcases real-world Selenium automation use cases and browser interaction workflows.

---

## ✨ Features

- 🌐 Automated browser interaction
- 🤖 WhatsApp automation bot
- 🔔 Push notification automation
- 🧪 Automated testing workflows
- 📄 JSON-based data handling
- ⚡ Cross-page interaction automation
- 📸 Automated screenshot testing
- 🖥️ Windows automation support

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/RussanaMary/Selenium.git

cd Selenium
```

---

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run any automation script using:

```bash
python filename.py
```

Example:

```bash
python whatsapp_self_bot.py
```
